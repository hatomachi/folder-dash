# Folder Dash (folder-dash)

## コンセプトと解決する課題
1つの機能エンハンスやタスク開発において、1つの専用ディレクトリを割り当てる**「1タスク＝1フォルダ」**の原則を強力にサポートするためのObsidianカスタムプラグインです。

本プラグインは、以下の現場の課題を解決します。
- **カオス化の防止**: Git管理下で、仕様書（成果物）、個人の壁打ちメモ、そしてレビュー指摘ノートなどが1つのフォルダに混在し、どれが最新のドキュメントで何が進行中なのか分からなくなる状態を防ぎます。
- **TOC（制約条件の理論）に基づくリードタイム管理**: 単純な「作業中」というステータス管理ではなく、「実際に手を動かした時間（プロセスタイム）」と「他チームの返答やレビューを待っていた時間（ウェイトタイム）」を明確に分離・計測し、開発の真のボトルネックを可視化します。
- **プロジェクト全体の俯瞰 (v2.1.0~)**: 個別タスクだけでなく、Epic（テーマ）単位で各タスクの進行状況をバックログダッシュボードから一元管理できるようにします。

## 主な機能
- **「タスクノート (`_Task.md`)」の自動生成と左ペイン完全連携**: プラグインを有効にすると左ペイン（サイドバー）に「Folder Dash」タブが常設されます。メインペインで開いているノートのフォルダを自動検知し、裏側に保存された `_Task.md` のデータを元にダッシュボードを自動生成・表示します。
- **自動分類ダッシュボード**: フォルダ内のファイルを「成果物」「レビュー指摘」「メモ」など（※設定から自由にカスタマイズ可能）へ自動分類し、最終更新日時が新しいものから順（降順）に詳細表示します。また、見出し横の「＋追加」ボタンから、適切なプロパティを持った新規ノートを瞬時に作成できます。左ペインの各リンクをクリックすれば、シームレスにメインペインでファイルが開きます。
- **アンドン（状況可視化）機能**: レンダリングされたフロントの機能ボタン（「▶ 着手」「⏸ ブロック」「✅ 完了」）を押すことで、現在のステータスと時間が自動記録されます。ブロック時にはなぜ作業がストップしたのかの理由を入力でき、その履歴（誰がいつどのアクションをしたか）が下部にタイムラインとして記録されます。
- **Git連携（担当者の自動取得）**: 煩わしい手動設定を避けるため、バックグラウンドのNode.jsレイヤーで `git config user.name` からユーザー名を自動取得し、ボタンアクション時の担当者（Assignee）や操作履歴に自動打刻・引き継ぎ（テイクオーバー）を行います。
- **バックログボード（カンバン / アジェンダビュー） (v2.1.0~)**: プロジェクト全体のタスクを俯瞰するためのメインダッシュボード画面です。「Kanban」と「Agenda」の2つのビューをリアルタイムで切り替えられます。
  - **Kanbanビュー**: ステータスごとにタスクを列で管理し、直感的にプロジェクト状況を把握できます。
  - **Agendaビュー**: 定例会議の読み合わせ用に最適化されたビューです。タスクはEpic（`_Epic.md`）ごとのアコーディオンにグループ化され、「概況」や「スケジュール」などのメタデータとともに行表示されます。全体を見渡せる「↕️ 一括開閉」ボタンも搭載。各タスクの `latest_update` や、Epic自体のメタデータもダッシュボード上で直接リッチ編集（赤字・青字等）でき、入力設定した内容は即座に再描画されます。また、「＋新規エピック作成」「＋タスク追加」などのボタンからダッシュボード上で直接Obsidian内にフォルダツリーを自動展開できます。
  - **デイリースクラム（朝会）向けフィルター**: 全タスクから自動抽出された「担当者（Assignee）」によるメンバーごとの絞り込みや、「🌟 今日やる」フラグが立ったタスクだけを表示するフィルター機能により、日々のタスク確認とミーティングを効率化します。

## 使い方
1. Obsidian上で新しいタスク・エンハンス用のフォルダ（または既存のフォルダ）を作成し、その中にあるノートを開きます。
2. 左ペインの「Folder Dash」ダッシュボード画面が表示されます（未作成の場合は「タスクノートを作成する」ボタンから初期化します）。
3. ダッシュボード画面にて「▶ 着手」ボタンを押して作業を開始します。
4. 別チームへ仕様の問い合わせをする際や、PRを出してレビューを待つ間は、「⏸ ブロック」ボタンを押します。
5. 出現した入力枠に理由（例: "レビュー待ち"）を記録し、作業に戻る際は再度「▶ 着手」を押します。
6. すべてが終わったら「✅ 完了」を押すことで、トータルのリードタイムと純粋な稼働時間が集計されます。
7. コマンドパレットから「Folder Dash: 現在のフォルダにタスクノートを作成/開く」や「Folder Dash: バックログボードを開く」を実行することで、素早くプロジェクト全体を俯瞰することができます。
